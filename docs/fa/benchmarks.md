# بنچمارک

در حال حاضر بنچمارک‌های ASV به صورت پیش‌فرض در مخزن وجود ندارند. اگر قصد دارید
عملکرد را دنبال کنید، می‌توانید در فورک خودتان ASV را برگردانید:

1) افزودن `asv` به وابستگی‌های اختیاری در `pyproject.toml`
2) افزودن `asv.conf.json` و پوشه `benchmarks/`
3) اجرای `uv run asv run --quick`

برای بررسی سریع عملکرد، می‌توانید از این اسکریپت استفاده کنید:

- `scripts/profile_conversion.py`
